Welcome to SentinelPi

Your secure, modular, AI-powered home surveillance system.

Features

  • Multi-camera support (USB + RTSP/IP)
  • AI detection with YOLOv5 Nano / TensorFlow Lite
  • Motion detection via MotionEye
  • Custom Flask overlay dashboard
  • MQTT alerts and Telegram notifications
  • Home Assistant integration
  • VPN support with WireGuard
  • Secure access with TLS
  • Full Docker and Ansible automation
